模块参数配置
1.【模块参数配置】函数定义 位于 addons/we7_demo/module.php
2.【模块参数配置】模板文件 位于 addons/we7_demo/template/rule.html，模板文件名称可随意命名。
3. 显示规则表单，自动调用函数 function fieldsFormDisplay() {} 用于附加额外字段。
4. 提交规则表单，在保存规则前调用 function fieldsFormValidate() {} 验证附加字段，验证成功保存规则字段，验证失败中断操作并提示错误信息。
5. 规则保存成功后，将调用 function fieldsFormSubmit() {} 完成入库操作，此处禁止使用 exit(); 或 die();。
